# Turismo-Mauá 🏝️

## Objetivo ✨
o objetivo deste projeto foi criar uma página web simples mostrando alguns pontos turísticos da cidade de Mauá-SP

## arquivos 🗄️
os arquivos html e css juntos criam a página completa com as cores e imagens. A pasta "Imagens" contém algumas fotos da página no navegador e as fotos utilizadas no projeto. Por fim, o arquivo "IA-Turismo" contém alguns registros de conversas com a IA ChatGPT para revisar o projeto.

## Linguagens 🧑‍💻
HTML, CSS e Framework

## como executar ▶️
Baixe o arquivo Turismo.html e estilo_turismo.css
Abra o arquivo html da pasta em seu navegador.
